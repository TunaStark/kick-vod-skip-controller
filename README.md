# 🟢 Kick.com Video Skip Controller

Are you tired of the default **30-second** skip on Kick VODs? This lightweight script allows you to customize the skip duration using your keyboard's arrow keys! 

### ✨ Features:
* **Customizable Duration:** Overrides the default 30s jump. Set it to 5s, 10s, or whatever you prefer (Default is 5s).
* **Native Integration:** Change the time directly from the Tampermonkey extension menu. No messy on-screen buttons.
* **Safe & Clean:** Zero external dependencies, no tracking, and 100% local execution.

### ⚙️ How to Change the Skip Time:
1. Open any Kick.com VOD.
2. Click on the Tampermonkey extension icon in your browser.
3. Click on **"⚙️ Set Skip Time (Current: 5s)"** under the script's name.
4. Enter your desired skip time in seconds and hit OK!

# 🟢 Kick.com Video Atlama Kontrolcüsü (Türkçe)

Kick yayın tekrarlarındaki (VOD) varsayılan **30 saniyelik** atlama süresinden (sağ/sol ok tuşları) sıkıldınız mı? Bu hafif script sayesinde atlama süresini dilediğiniz gibi özelleştirebilirsiniz!

### ✨ Özellikler:
* **Özelleştirilebilir Süre:** Varsayılan 30 saniyelik atlama sınırını kaldırır. 5s, 10s veya istediğiniz bir değere ayarlayın (Varsayılan 5s).
* **Temiz Arayüz:** Süreyi doğrudan Tampermonkey eklenti menüsünden değiştirebilirsiniz. Sayfada çirkin butonlar kalabalık yapmaz.
* **Güvenli ve Hafif:** Dış sunucu bağlantısı veya veri takibi (tracking) kesinlikle içermez. Sadece DOM üzerinde %100 yerel ve güvenli çalışır.

### ⚙️ Süreyi Nasıl Değiştiririm?:
1. Herhangi bir Kick yayın tekrarını açın.
2. Tarayıcınızdan Tampermonkey eklentisi simgesine tıklayın.
3. Bu scriptin altındaki **"⚙️ Set Skip Time (Current: 5s)"** seçeneğine tıklayın.
4. İstediğiniz saniyeyi yazın (örn: 5, 10) ve onaylayın! Oynatıcı anında güncellenecektir.
